 Expense Tracker App using Data Science

An end-to-end Python application that uses synthetic financial data to simulate expense tracking, categorical analysis, and trend forecasting.

 Features
-Synthetic Data Generation:Generates 500+ realistic financial transactions.
-Data Cleaning:Handles date formatting and categorization automatically.
-Interactive Dashboard:Built with Streamlit and Plotly for real-time filtering.
-Feature Engineering:Extracts patterns based on days of the week and months.

 Tech Stack
-Language: Python 3.x
-Libraries: Pandas, NumPy, Plotly, Streamlit
-Environment: Virtualenv

 Project Structure
- main.py: Entry point for the Streamlit dashboard.
- src/: Contains data generation and processing logic.
- requirements.txt: List of dependencies.

 How to Run
1. Clone the repository.
2. Install dependencies: pip install -r requirements.txt
3. Run the app: streamlit run main.py
